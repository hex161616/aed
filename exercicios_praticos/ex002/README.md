## Descrição do Exercício dado em Aula

Dado o primeiro elemento de uma lista encadeada, gire a lista para a direita em k posições.

## Casos que foram testados

Example 1:

Input: head = [1,2,3,4,5], k = 2
Output: [4,5,1,2,3]

Example 2:

Input: head = [0,1,2], k = 4
Output: [2,0,1]
